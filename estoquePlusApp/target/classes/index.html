Reflexões e Aprendizados

Igor F. Monteiro | Engenharia de Computação - Instituto Infnet

Objetivos de Ensino

A jornada acadêmica no curso de Engenharia de Computação na Faculdade Infnet foi projetada para equipar os estudantes com um profundo conhecimento teórico e prático nas áreas de desenvolvimento de software, redes de computadores, e sistemas embarcados. Almejamos desenvolver soluções inovadoras e eficientes para problemas complexos do mundo real.

Capacidades Adquiridas

  • Proficiência em diversas linguagens de programação e frameworks, incluindo Java e Spark Java.
  • Capacidade de desenvolver APIs RESTful eficientes e escaláveis.
  • Habilidade em configurar e gerenciar bancos de dados, simulando operações CRUD.
  • Competência no uso de ferramentas de versionamento, como Git, para colaboração e gestão de código.
  • Desenvolvimento de interfaces de usuário intuitivas utilizando HTML, CSS e JavaScript.

Preparação para o Mercado

Através do curso na Infnet, adquiri não apenas conhecimento técnico, mas também habilidades interpessoais essenciais para a inovação e liderança no setor de tecnologia. Estou preparado para enfrentar os desafios do mercado com uma abordagem analítica e criativa, prontamente adaptável às necessidades em constante evolução da indústria.
